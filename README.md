# 实时经济走势系统

这是一个部署到 GitHub Pages 的实时市场观察系统。

功能：

- 跟踪美股、A股/港股、虚拟货币、黄金/贵金属、原油、美元指数、美债收益率和人民币汇率。
- 每小时通过 GitHub Actions 自动刷新行情和新闻。
- 输出交互式网页、中文简报和 JSON 快照。
- 手机 ChatGPT 可以读取固定 HTTPS 链接，不依赖电脑是否开机。

主要链接：

- `index.html`：交互式图表面板。
- `latest_market_brief.md`：中文市场简报。
- `latest_snapshot.json`：完整结构化快照。
- `market_data.js`：网页图表数据。

风险提示：

本系统只做信息整理、情景推演和风险提示，不构成个性化投资建议。实际交易前应结合资金期限、风险承受能力、仓位、税费和监管要求。
